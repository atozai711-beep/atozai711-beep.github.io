# Dev Butani's Portfolio Website

A minimal, professional website hosted on GitHub Pages for Android app developers.

## Contents

- **index.html** - Main homepage with navigation, hero section, about, apps showcase, and contact
- **privacy-policy.html** - Comprehensive privacy policy for your applications
- **styles.css** - Modern, responsive styling with mobile optimization
- **ads.txt** - Advertising configuration file for Google ads

## Features

✅ **Fully Responsive** - Works perfectly on mobile, tablet, and desktop  
✅ **Modern Design** - Clean, minimal aesthetic using system fonts  
✅ **Fast Loading** - Optimized CSS and no unnecessary dependencies  
✅ **SEO Friendly** - Proper meta tags and semantic HTML  
✅ **Easy to Customize** - Simple structure, easy to update app listings  

## How to Use

1. Clone or fork this repository
2. Edit the files to add your app information
3. GitHub Pages will automatically host it at `https://atozai711-beep.github.io`

## Customization

### Adding Your Apps

Edit the apps grid section in `index.html`:

```html
<div class="app-card">
    <h3>Your App Name</h3>
    <p>Your app description</p>
    <a href="https://play.google.com/store/apps/details?id=com.example" class="btn btn-small" target="_blank">View on PlayStore</a>
</div>
```

### Update Contact Information

Replace email addresses and developer information throughout the files:
- `index.html`
- `privacy-policy.html`

### Customize Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #007AFF;
    --secondary-color: #f2f2f7;
    /* ... more variables */
}
```
